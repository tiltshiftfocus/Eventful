# Eventful
An Android app to keep track of history of events.
A timer is shown in each event such that it will let you know how long has it been since the event.

(Great for tracking warranties and memories you wish to remember.)

## Ideas
Grabbed from iOS app "Time Flies" and
Android "Time Flies" by pt (https://play.google.com/store/apps/details?id=fi.zt.timeflies)
